This is an example of a set of help page created using the Advanced help framework. If you have the appropriate permissions, you may view the contents of this and other HTML help files.

Note: The module has still a number of regressions compared to the Drupal 7 versions. This are tracked in this issue in the project's issue queue: [meta] Regressions vis-à-vis Drupal 7. Sections referring to such regressions are marked "[D7 regression]". These are kept as placeholders, but the features they describe are not yet available.

To view it as a book, navigate to Manage » Help » Advanced help » Advanced help example.

To view help shown in popups navigate to Manage » Example help and [D7 regression ] and click on the question mark icons.

[D7 regression] To try out the search features, first enable advanced help search, then run cron to index the help pages.
